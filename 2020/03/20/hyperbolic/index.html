双曲函数的性质及恒等变换

发表于 2020-03-20 分类于 数学

前言

三角函数用圆定义, 而双曲函数用双曲线定义, 它们之间存在丰富的联系. 双曲函数在物理中有较广泛的应用, 而它本身也是一类性质良好的初等函数.
本文介绍双曲函数的定义、性质和恒等变换.

1. 双曲函数的定义

双曲函数虽然是个初等函数, 但它的定义并不初等.
双曲函数基于双曲线 $x^2-y^2=1$ 的右支定义. 设一直线过原点和双曲线右支上的一点 $(x_0,y_0)$, 称该直线、双曲线和 $x$ 轴围成的面积的 $2$ 倍为双曲角. 若 $y<0$, 则双曲角取负值. 双曲角即双曲函数的自变量.

事实上, 若不考虑三角函数的周期性, 在单位圆中有 $\displaystyle S_{扇形}=\frac{lR}{2}=\frac{l}{2}$, 那么扇形的面积的 $2$ 倍也即三角函数的自变量. 由此可以发现双曲函数和三角函数定义的相似性.

然后设双曲角为 $x$, 定义双曲正弦函数 $\sinh x = y_0$, 双曲余弦函数 $\cosh x = x_0,$ 双曲正切函数 $\displaystyle \tanh x= \frac{y_0}{x_0},$ 以及其余的三个双曲函数.

积分可得, 双曲角的值为 $\ln(x_0+y_0)$, 于是我们可以得到双曲函数的初等表达式:

$\displaystyle\sinh x=\frac{e^x-e^{-x}}{2},\cosh x=\frac{e^x+e^{-x}}{2},\tanh x=\frac{e^x-e^{-x}}{e^x+e^{-x}}$

2. 双曲函数的性质

上面三类双曲函数的定义域都是 $\mathbb{R}$.

  • $y=\sinh x$ 的值域是 $\mathbb{R}$, 在定义域上单调递增, 是奇函数.
  • $y=\cosh x$ 的值域是 $[1,+\infty)$, 在 $[0,+\infty)$ 上单调递增, 是偶函数.
  • $y=\tanh x$ 的值域是 $[-1,1]$, 在定义域上单调递增, 是奇函数.

3. 恒等变换

基本关系

双曲函数的和差

直接变形即可证明, 比如
$\begin{aligned}
\sinh(x+y)&=\frac{e^{x+y}-e^{-(x+y)}}{2}\\
&=\frac{(e^{x+y}+e^{x-y}-e^{-x+y}-e^{-(x+y)})+(e^{x+y}-e^{x-y}+e^{-x+y}-e^{-(x+y)})}{4}\\
&=\frac{(e^x-e^{-x})(e^y+e^{-y})+(e^x+e^{-x})(e^y-e^{-y})}{4}\\
&=\sinh x\cosh y+\cosh x\sinh y
\end{aligned}$.
其他的公式可以使用以上两组公式简单推导得到, 这里不再列举.

4. 反双曲函数

  • $y=\operatorname{arsinh}x=\ln\left(x+\sqrt{x^2+1}\right),$ 定义域和值域都为 $\mathbb{R}.$
  • $y=\operatorname{arcosh}x=\ln\left(x+\sqrt{x^2-1}\right),$ 定义域为 $[1,+\infty)$, 原函数是偶函数, 反函数值域为 $[0,+\infty).$
  • $\displaystyle y=\operatorname{artanh}x=\frac{1}{2}\ln(\frac{1+x}{1-x}),$ 定义域为 $(-1,1),$ 值域为 $\mathbb{R}.$
